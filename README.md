# 网易云专辑封面下载器

![Python Version](https://img.shields.io/badge/python-3.6+-brightgreen)
![License](https://img.shields.io/badge/license-MIT-blue)

一个简单高效的网易云音乐封面下载工具，自动获取高清封面并按专辑整理。

## ✨ 特性
- **高清封面**：智能获取高质量封面（最高9999×9999）
- **自动整理**：按"艺人 - 专辑"创建文件夹结构
- **简单易用**：单文件运行，无需复杂配置
- **批量就绪**：支持连续下载多个封面
- **格式优化**：优先选择WebP等高效格式
