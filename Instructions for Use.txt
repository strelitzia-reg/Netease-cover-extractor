网易云专辑封面下载器 使用说明

【快速开始】
1. 确保已安装 Python 3.6 或更高版本
2. 双击运行 "运行.bat" 文件
3. 按提示输入网易云音乐链接

【支持的链接格式】
• 歌曲链接: https://music.163.com/#/song?id=1901371647
• 歌曲链接: https://music.163.com/song?id=3326310773
• 专辑链接: https://music.163.com/#/album?id=151973204
• 直接输入数字ID: 1901371647

【文件保存位置】
所有封面自动保存到 "网易云专辑封面" 文件夹
按 "艺人名 - 专辑名" 创建子文件夹

【常见问题】
1. 如果双击.py文件闪退？
   → 请双击"运行.bat"文件启动

2. 提示"缺少requests库"？
   → 运行: pip install requests

3. 下载失败或显示"未知艺人"？
   → 可能是特殊歌曲，尝试其他链接

4. 如何批量下载？
   → 目前支持单次下载，可多次输入链接
